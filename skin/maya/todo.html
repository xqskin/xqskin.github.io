import maya.cmds as cmds
import re
for jnt in cmds.ls(sl=1,type='joint'):
    get_attr=cmds.listConnections((jnt + ".worldMatrix"),p=1)
    get_sc=cmds.listConnections(jnt + ".worldMatrix")
    match = re.search(r'\[(\d+)\]', get_attr[0])
    if match:
        get_index = int(match.group(1))
        cmds.connectAttr("{0}.parentInverseMatrix".format(jnt), "{0}.bindPreMatrix[{1}]".format(get_sc[0],get_index), f=1)

路径动画的几种方法 ​

  • curve wrap 先选择模型、再选择线 curve wrap 上的 offset属性

  • Mash 选择多边形 再Mash菜单下create mash network 然后点击Mash editor 添加Curve 按住鼠标中键将曲线从大纲拖到Input Curves上 调节MASH1_Distribute的Distance X值 可以改变间距 然后通过调整刚添加的Mash1_Curve节点的Clip start属性即可

  • spline IK 创建Spline IK 取消勾选auto create curve 先选择骨骼链第一个 再选择骨骼链最后一个 然后按住shift加选曲线 通过调整ikhandle1上的offset值

  • motion path 先选择模型、再选择线 attach to motion path 然后再选择模型 再选择 Flow path Object 然后调整分段数

  • nurbs -extrude 先选择圆环曲线 再选择路径曲线 然后选择Surfaces菜单下面的Extrude 前面四个选项都选择最后一个 倒数第二个选择最后一个 其他保持默认 通过调整subcurve2节点的Max Value 值来达到效果

  • 还有一种是笔刷的 content broswer 里面 选择Paint Effects 里面的 FunMesh 在strokeShapeSpackle1里面的End Bounds里面 调节 Max clip的值即可

Maya拓扑 ​

激活被拓扑模型 打开建模工具包——四边形绘制 ctrl+shift+Q 进入拓扑模式

在模型上点击会生成绿点 创建4个点 按shift点击中间会生成面

光标停留在点、线、面 上呈红色的时 为选中状态 在边上按 Tab 可以挤出 挤出的面与附近的点较近时会自动合并 （显示extrude 挤出） Tab+中键 可以按 循环边挤出 Ctrl+左 添加循环线 Ctrl+shift+左 删除
Shift+左 松弛布线 可以配合软选择使用 B键 工具：
多切割工具 和建模模式下功能一致
清除点 清除掉多余的绿点 可以在热盒中选择使用 Tab+右 激活约束选项 ：可以设置线框、网格属性

三个提升拓扑效率技巧： ​

  • 模型面数过高导致卡顿严重 使用GPU缓存 缓存——GPU缓存——导出当前选择 缓存时间范围：当前帧 导出模型 格式：abc 此时的模型 没有面数显示 再导入模型 缓存——GPU缓存——导入 遇问题：缓存里没有GPU缓存 首选项也没有GPU缓存设置 解：在插件管理器中 找到gpuCache.mll 并加载

  • 关闭历史记录 拓扑结束后记得恢复

  • 拓扑过程中穿模问题 使用工具包中的 曲面偏移功能 建议数值：0.1

拓扑过程中可以切换模式合理利用不同工具，进行有效调整
