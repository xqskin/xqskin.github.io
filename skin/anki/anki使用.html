Anki使用备忘 ​

WARNING

不贪多，不嫌少。一定要养成最小闭环的学习习惯！一定要养成最小闭环的学习习惯！一定要养成最小闭环的学习习惯！重要的事情说三遍！就是每天不论时间多、时间少都要坚持“学习-思考-记录-复习”的流程，时间多就进行大闭环，时间少就进行小闭环，但一定要把当天学的记下来，当天记的复习了。题干一定要简洁，做好知识的相对最小化。每天一定要坚持完成复习任务，宁愿不学新，也要复习旧。

WARNING

以前上学时没有人会拿着还没上过的教材直接就开始背诵，但用ANKI时就很常见，很多人会把没经过学习的材料整理到ANKI里复习。实际上，要想真正掌握学过的知识，我们需要先学习，然后将自己的思考成果以笔记的形式巩固下来，再尝试复述（背诵），当勉勉强强达到背诵（Recite）效果以后，再使用ANKI复习（Review）。总之，没有背诵，何来复习？

使用#号表示标题层级 前三级作为文献信息 会自动串起来，作为书名号字段的内容 四级作为问题、五级作为答案，多个答案就多个五级 左右各一个星号用来表示斜体，到了ANKI里会以高亮显示 左右各两个星号加粗 图片使用图床链接的形式插入

- 填空题的本质就是在需要挖空的字符两边加上了{{c1::挖空}},word中替换 {{c1::^&}}
- 填空题可以增加提示项 {{c1::975::谐音"就气我"}}
- ：是否显示、以及点击显示	1、通过字段是否存在来显示{{#field}} text or field here{{/field}} 2、点击显示字段{{hint:fieldname}}	{{#code}} Code Sample： {{/code}} {{hint:}}

TIP

挖多个空同时显示的方法，和挖多个空逐个显示的方法，稍有不同。前者需要让同时出现的数值相同，如c2、c2是两个空一起出现，c3、c3、c3是三个空一起出现。一般情况下按挖空键或按钮，数字是递增的，如c1、c2、c3、c4…这样，如果希望不递增，就要多按住一个alt键。也就是4个按键，看起来挺考验动手能力，其实适应了就很快。

TIP

连续挖空多次，就得到这样的填空题。三个空，依次出现。如果你没做过修改，这三张卡片会分布在3天里出现，而不是连续出现。如上图所示，第一天只出现第一个空。为什么其他空不出现呢？这涉及到一个概念“相关卡片”，因为一句话上挖出来的三个空，每次学习一个空就会把整句话读一遍，就会对第二个空的学习造成提示，会导致我们高估自己的熟悉程度。所以，分布在不同天学习，降低了这种相互提示。如果想取消这种“搁置”，可以打开学习方案设置，在“新卡片”和“复习”两个标签页下取消“搁置相关新卡片到隔日”和“搁置相关复习卡片到隔日”。

TIP

文本文档另 存 为 UTF-8的编码方式 把从Excel里复制的内容粘贴到这个Text文档里，保存，关闭

TIP

分隔符\t 如果你在测试时反复使用相同内容，再次导入时会被当成重复内容忽略

TIP

iOS版的入口：卡片复习界面-右上角tools-More-Edit Card Template 安卓卡片类型(模板)编辑入口: 牌组(主页) - 点击右上角(菜单) - 笔记类型

TIP

在将逐个显示（多张卡）改成同时显示（单张卡）时，会遇到一个问题，即复习到原来的卡片时，会提示当前卡片为空，需要你清理空卡片、检查数据库。这是正常的，并不是你的笔记出错了。原因在于Anki为了防止误删笔记，在修改了填空题的个数后，暂时保留了缓存。用户只要到设置了找到“检查数据库”和“清理空卡片”就可以了。

TIP

公式只能用图片，或截图拍照，或用编辑器，用编辑器的话最终形式也还是图片。

TIP

在完成前面的替换以后，还要手动删除这些已经不再使用的图片。如下图所示，这里呈现出来的都是不再被使用的图片残留，可以继续点击右下角的“删除未使用的文件”进行删除。
