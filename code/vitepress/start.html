文章目录

准备 ​

  • nodejs 18.14+

安装 pnpm ​

npm install -g pnpm

查询 pnpm源 ​

pnpm get registry

设置pnpm淘宝源 ​

pnpm config set registry https://registry.npmmirror.com/

下载vitepress ​

pnpm add -D vitepress

初始化 ​

pnpm vitepress init

文件目录 ​

启动项目 ​

pnpm run docs:dev

前端预览 ​

首页装饰 ​

定义favcion() ​

  head: [["link", { rel: "icon", href: "/logo.svg" }]],

 footer:{copyright:"Copyright@ 2024 xqskin.com"},

内页两栏布局 ​

增加themeConfig设置 ​

sidebar: false, // 关闭侧边栏
aside: "left", // 设置右侧侧边栏在左侧显示
